AstronomyService(String, String)

The astronomy service can be used retrieve rise, set, noon and twilight times for sun and moon for all locations. The service also exposes the azimuth of the events and altitude and distance (for the noon event).

Radius

Search radius for translating coordinates (parameter placeid) to locations. Coordinates that could not be translated will yield results for the actual geographical position – if you would like to query for times at an exact location, specify a radius of zero (0).

Declaration
public int Radius { get; set; }
Property Value
Type Description
System.Int32

The radius in kilometers. Default is infinite, but only locations within the same country and time zone are considered.
